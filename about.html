    • Known Issues

  • The code window does not automatically scroll to the current value of PC.
  • The code window is not updated if the program overwrites code (self-modifying code). No warning is emitted either.
  • The simulator will not detect all forms of illegal input.
  • Assembling the chosen file after modifying that file does not work. Need to reselect the file by pressing the 'Choose File' button.
  • The simulator may not work on browsers or networks that disable JavaScript or file operations. The simulator does not provide any feedback or error messages when this occurs.
